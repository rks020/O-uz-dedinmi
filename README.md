# Ödedimmi Clone

Bu proje Flutter ile geliştirilmiş bir Gelir Gider Takibi uygulamasıdır.

## Özellikler

- **Gider Takibi**: Harcamalarınızı listeleyin, kategorilere ayırın.
- **Analiz**: Harcamalarınızı grafiklerle görselleştirin.
- **Modern Tasarım**: Şık arayüz ve kullanıcı dostu deneyim.

## Kurulum

1. Proje dizininde terminali açın.
2. Bağımlılıkları yükleyin:
   ```bash
   flutter pub get
   ```
3. Uygulamayı çalıştırın:
   ```bash
   flutter run
   ```

## Kullanılan Paketler

- `flutter_riverpod`: State management
- `fl_chart`: Grafikler
- `google_fonts`: Yazı tipleri
- `font_awesome_flutter`: İkonlar
- `intl`: Tarih ve para birimi formatlama
